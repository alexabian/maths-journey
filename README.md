# Maths Journey

A calm, offline-first Year 2 maths adventure game for GitHub Pages.

## Current vertical slice

- Home Village and Number Forest
- Professor Owl lesson flow
- Teach → demonstrate → guided practice → independent practice → spaced review
- Gentle hints and misconception-friendly feedback
- XP, coins, streak, lesson completion and village restoration
- Local progress persistence
- Parent progress view behind a simple gate
- No accounts, analytics, ads or network dependency

## Run locally

```bash
python3 -m http.server 4173
```

Open http://localhost:4173.

## GitHub Pages

The included `CNAME` targets `journey.estructura.studio`. Enable GitHub Pages from the repository's `main` branch root, then configure the DNS record at the domain provider as a CNAME to the repository's GitHub Pages hostname.
